(device-height)

当您的设备分辨率有高度时，本段落背景色为绿色。

(device-height:1050px)

当您的设备分辨率高度 = 1050px时，本段落背景色为绿色。

(min-device-height:600px)

当您的设备分辨率设置高度 => 最小值600px时，本段落背景色为绿色。

当您的设备分辨率设置高度 < 最小值600px, 本段落无背景色

(max-device-height:1100px)

当您的设备分辨率设置高度 <= 最大值1100px时，本段落背景色为绿色。

当您的设备分辨率设置高度 > 最大值1100px, 本段落无背景色

(min-device-height: 600px) and (max-device-height:1100px)

最小值600px <= 您的设备分辨率设置高度 <= 最大值1100px时，本段落背景色为绿色。

其它高度, 本段落无背景色
