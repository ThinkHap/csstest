(device-width)

当您的设备分辨率有宽度时，本段落背景色为绿色。

(device-width:1680px)

当您的设备分辨率宽度 = 1680px时，本段落背景色为绿色。

(min-device-width:400px)

当您的设备分辨率设置宽度 > 最小值400px时，本段落背景色为绿色。

当您的设备分辨率设置宽度 < 最小值400px, 本段落无背景色

(max-device-width:800px)

当您的设备分辨率设置宽度 < 最大值800px时，本段落背景色为绿色。

当您的设备分辨率设置宽度 > 最大值800px, 本段落无背景色

(min-device-width: 400px) and (max-device-width:800px)

最小值400px < 您的设备分辨率设置宽度 < 最大值800px时，本段落背景色为绿色。

其它宽度, 本段落无背景色
