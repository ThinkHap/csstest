(resolution)

定义设备的分辨率，即像素密度。如：96dpi, 300dpi, 118dpcm。

用整数值来定义。不允许负值

当您的输出设备支持resolution时，本段落背景色为绿色。

(min-resolution:18dpcm)

当您的输出设备的分辨率 >= 18dpcm 时，本段落背景色为绿色。

否则, 本段落无背景色

(min-resolution:96dpi)

当您的输出设备的分辨率 >= 96dpi 时，本段落背景色为绿色。

否则, 本段落无背景色

(max-resolution:300dpi)

当您的输出设备的单色帧缓冲器中每像素的位数 <= 300dpi时，本段落背景色为绿色。

否则, 本段落无背景色

(min-resolution:96dpi) and (max-resolution:300dpi)

当 96dpi <= 您的输出设备的单色帧缓冲器中每像素的位数 <= 300dpi 时，本段落背景色为绿色。

否则, 本段落无背景色
